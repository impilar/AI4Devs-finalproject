# Skill: create-user-story

Genera una historia de usuario individual.

## Inputs

| Rol | Ruta |
|-----|------|
| User Story Map | `docs/product/user-story-map/user-story-map-v1.md` |
| Plantilla | `knowledge/templates/product/user-story_template.md` |

## Output

`docs/product/user-stories/US-NNN-titulo-corto.md`

## Formato

> Como [persona], quiero [acción], para [valor]

Incluir criterios de aceptación Gherkin y release (MVP/V1/V2+).
