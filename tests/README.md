# Tests

Estrategia de pruebas transversales (E2E, integración, rendimiento).

| Tipo | Carpeta | Herramienta |
|------|---------|-------------|
| E2E | `e2e/` | Playwright |
| Integración | `integration/` | Vitest + Supertest |
| Rendimiento | `performance/` | k6 |
| Fixtures | `fixtures/` | Datos compartidos |

Los tests unitarios viven junto al código en `src/`.
